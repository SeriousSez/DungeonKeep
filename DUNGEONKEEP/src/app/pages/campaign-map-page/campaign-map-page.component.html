Field Notes

How to Use This Map

The editor now supports independent vision and movement walls, remembered sight, and live token-based visibility. This panel summarizes the parts that affect table play the most.

Tools

What each mode does

  • Select: inspect and move drawings, landmarks, terrain, labels, and tokens.
  • Draw: add freehand lines for paths, borders, hazards, notes, or callouts.
  • Walls: draw walls that can block sight, movement, or both once selected.
  • Erase Lines: trim only the touched section of a drawing instead of deleting the whole mark.
  • Place tools: landmarks, terrain, labels, and tokens each stay active until you stop placing.

Shortcuts

Fast editing controls

  • Shift while drawing: snap new points to grid intersections.
  • Shift with a selected token: temporarily preview that token's vision.
  • Ctrl+Shift while drawing or placing walls: add straight-line corners that stay snapped to the grid until you release Ctrl.
  • Keys 1-8: switch tools in order from Select, Draw, Walls, Erase Lines, Landmarks, Terrain, Labels, to Tokens.
  • Delete or Backspace: remove the selected drawing, label, landmark, or token.
  • Ctrl/Cmd+Click: instantly delete a selected drawing target.
  • Enter or Escape: confirm or close the active confirmation dialog.

Vision and Walls

How visibility behaves

  • Selected walls: can toggle sight blocking and physical blocking separately.
  • Token movement: respects walls marked for movement, so tokens cannot pass through them.
  • Preview Sight: still gives owners a manual way to inspect one token at a time.
  • Remembered areas: remain visible after a token has already explored them on this board.

Interaction

What to expect on the board

  • Click once to select an item before editing its details.
  • Drag in Select mode to reposition drawings, landmarks, terrain, labels, and eligible tokens.
  • Drawing and erasing work directly on the board, while labels and terrain use the right sidebar to set options before placement.
  • The guide, grid, wall, and fullscreen controls are view helpers and do not change map data on their own.

Saving

How changes are kept

  • Use Undo and Redo while editing to recover recent map changes.
  • Your edits remain local until you save the map, though active autosave will push map changes quickly.
  • Generated map art clears existing drawings but keeps landmarks unless you remove them yourself.
  • Remembered sight is stored per explored token on the current board until you reset it.
  • Only campaign owners can edit the board, while members can still inspect the current map state.

Table Guide

How to Read This Map

Player view now keeps assigned token vision active automatically, including remembered exploration. Use this as a quick reminder of what is visible and interactive at the table.

Legend

What the layers mean

  • Lines: paths, borders, hazards, callouts, or other hand-drawn notes.
  • Hidden wall settings: the owner can place invisible blockers that still affect line of sight, movement, or both.
  • Landmarks: important locations, objectives, warnings, and notable sites.
  • Terrain: mountains, forests, hills, reefs, caves, and wards that shape the scene.
  • Labels: region names and feature names placed directly on the board.

Viewing

What you can do here

  • Inspect the current board state without changing the owner’s layout.
  • Select tokens and review labels, landmarks, and linework visually.
  • Your assigned tokens keep their vision visible automatically while you are on the board.
  • Remembered explored areas stay partially revealed as those tokens move.
  • Toggle fullscreen or open this guide for a cleaner table view.
  • Use the layer controls to simplify the board when you need to focus on one visible element type.

Editing Access

When the board changes

  • Campaign owners can switch into Map Editor to add drawings, landmarks, terrain, and labels.
  • Assigned players may still be able to move specific tokens on the active map.
  • Movement-blocking walls apply to player-controlled token movement as well as sight.
  • Unassigned drawings and map decorations stay controlled by the owner.
  • If you need structural changes, move into editor mode instead of viewer mode.

Table Use

Good ways to use it live

  • Keep fullscreen open for encounters or overland travel reference.
  • Use labels and landmarks as anchors while the party moves tokens.
  • Use selected-token vision checks when the table needs to inspect one creature’s sightline in detail.
  • Treat linework as flexible markup, not only roads or routes.
  • Hide layers when you want a cleaner scene or less visual noise.
